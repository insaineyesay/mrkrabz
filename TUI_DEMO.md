## Key Features Shown

1. **Search Box (Top)**
   - Cyan colored border
   - Shows your query with cursor
   - Type naturally and press Enter

2. **Results List (Middle)**
   - Shows repository full name in bold
   - Star count in yellow with â­ emoji
   - Language in blue with ğŸ’» emoji
   - Selected item has â–¶ symbol and gray background
   - Scrollable with arrow keys

3. **Details Panel (Bottom-Middle)**
   - Shows full description of selected repo
   - Star, fork, and language stats
   - Clickable URL (opens when you press Enter)

4. **Help Bar (Bottom)**
   - Always visible keyboard shortcuts
   - Green for action keys, red for quit

## Color Scheme

- **Cyan**: Headers, URLs, query text
- **White/Bold**: Repository names
- **Yellow**: Star counts
- **Blue**: Languages
- **Green**: Fork counts, action keys
- **Gray**: Dimmed/unselected items
- **Red**: Error messages, quit key
- **Dark Gray**: Selected item background
